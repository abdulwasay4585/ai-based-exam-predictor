# AI Exam Question Prediction System

This core repository contains the decoupled services for the AI Exam Prediction platform.

## Architecture
- `backend/`: FastAPI serving endpoints
- `frontend/`: UI interface
- `ml_pipeline/`: PyTorch models, NLP components, and training scripts
- `data/`: Datasets (ignored from git)
- `saved_models/`: Trained model weights (ignored from git)
